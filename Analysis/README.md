# SingleProbeTimelapse
Resources to analyze pump-probe images that have multiple times in a single shot

See application in publication here (Peer-review in progress):

Twardowski, J., Kuz, C., Noor, M. Y., Lee, A., Leicht, J., Slipchenko, M., & Chowdhury, E. (2025). Single shot multi-time frame imaging of plasmas with a frequency tagged GHz pulse train (arXiv:2509.18336). arXiv. https://doi.org/10.48550/arXiv.2509.18336
